1.Comunicação Cliente X Servers - Módulo 01

Historia da WEB

Voltar

Em torno de 1944 á 1946 entrou em ação a 1º Geração de computadores , normalmente para operações cientificas e militares. criado o Havard Mark para calculos mais complexos, 1946 foi fabricado o Colossus utilizado pelos Alemãs e o Eniac utilizados pelos Americanos ambos para cripto análises e aumentar a segurança militares. De 1959 á 1965, entrou no mercado a 2º Geração de computadores aonde o tamanho dos computadores diminuiram um pouco e deixando os mesmos mais portateis. Depois de um periodo foi a fez dos computadores da 3º Geração aonde começou ser utilizados os circuitos integrados ( transitor/ Capacitores e Resitores).

O que são Clients

Voltar

Clients ( Clientes ) é o lado que os dispositivo consomem da informação da rede da internet, por exemplo Notbook / PC, a informação que é recebida do servidor depois de processados ( html/CSS/ Javascript/ Imagens). Precisamos entender que o lado Clients utiliza recursos de varios dispostivos e navegadores.

As linguagens client-side são linguagens onde apenas o seu NAVEGADOR vai entender.

O que são Servers

Voltar

Servers (servidores)podem ser acessivel pela internet e pela rede local privada (empresa ) atraves de um computador central,eles tende a ser equipamentos mais resitentes e fortes ( Custo $ elevados). Tipos de servidores:

  • Arquivos
  • Segurança (Firewall)
  • Streaming
  • E-mail
  • WEB
linguagens server-side são linguagens que o SERVIDOR entende. Isso quer dizer que vai escrever um código onde o servidor vai processá-lo e então vai mandar para o seu navegador a resposta

Estrutura básica do HTML

Voltar

Um documento HTML válido precisa seguir obrigatoriamente a estrutura composta pelas tags html , head e body e a instrução ! DOCTYPE.

Falando sobre Tags

Voltar

Os elementos HTML ou também chamados de tags HTML, são utilizados para informar ao navegador que tipo de estrutura é essa que está sendo construída, podendo ser títulos, parágrafos, imagens, links, entre outros

Atributos básicos

Voltar

Há dois tipos de atributos no HTML, os globais que são aceitos por todas as tags, como por exemplo: class, id, lang, style e algumas outras que você pode conferir na documentação também existem os específicos que somente algumas tags possuem, como src, disabled, href, label.

Textos

Voltar

Textos basicamente é o que o usuário Lê no Site criado, algumas tipografia (Tags) são utilizados para formatação de textos:

  • h1 h1 - Significa Header ( Cabeçalho ou Titulo);
  • h2/h3 - Utilizado para escrever Subtitulo;
  • p - Utilizado para escrever paragráfo;
  • blockquote - usado para citações em um paragrafo;
  • strong - serve para maerca uma palavra ou uma frase em negrito;
  • u (Underline) - tag para sublinhar uma frase;
  • i - tag para deixar a frase em conceito itálico;
  • mark - utilizado para marca o texto
  • sup - maracdor elevador - exemplo ( Tecnologia*)
Acima algumas tiprográfias mais utlizadas no sistema HTML.
